# taskflow-app
Application Fullstack pour gerer des projets, développée avec Next.js, React, Prisma, Clerk Auth, TailwindCSS et DaisyUI, Quill Editor . Permet la création, assignation, gestion, filtrage et clôture de tâches en temps réel. Inclut un système de rôles, une gestion de projet, une interface moderne et une expérience utilisateur fluide.
